# dev_task

A simple yet feature-rich Flutter-based personal task management app designed to meet all the requirements of a job opening in Singapore.

Why this project?
Familiar — task management is an easy-to-understand domain; there’s no need to learn a new domain while learning Flutter.
Scalable — can start very simply, then add features one by one.
Relevant — CRUD, auth, notifications, etc., are features found in almost all real-world apps.
